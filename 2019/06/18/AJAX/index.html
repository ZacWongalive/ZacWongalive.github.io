AJAX

  1. ajax简介

    1. Ayschronous JavaScript and XML(json格式)异步JavaScript和xml，ajax是属于客户端和服务端交互的一项技术
    2. 原有交互模式(整个页面刷新+请求和响应同步处理)：浏览器发送请求–>tomcat服务器–>tomcat将相应信息给浏览器–>浏览器显示响应结果 ajax交互模式(局部刷新+异步处理)：浏览器发送请求–>XMLhttpRequest发送–>tomcat服务器–>tomcat返回结果—>xmlhttprequest接受结果–>将结果显示在浏览器上

  2. ajax的好处

    1. 具有局部刷新和异步处理的特性
    2. 提高了程序交互效率(传递数据量少)
    3. 提高用户的体验（避免的页面的频繁刷新）

  3. ajax技术

    1. ajax技术是一系列技术的综合运用
    2. 它是基于js发请求和响应处理的
    3. 以XMLHttpRequest对象为核心
    4. 涉及html,css等渲染技术
    5. 涉及xml，json等数据交互格式

jquery当中使用ajax

  1. $.ajax() 详细看jq.jsp

    1. 这个函数将XMLHttpReqeust对象封装了
    2. 使用方式： $.ajax({url:"地址",type:”get/post”,data:”(提交的数据){"name":zac,"age":18},sucess:（里面写回调函数）function(msg){....},async:默认为true,true表示异步处理dataType:预期服务器返回的数据类型 xml,html,script,json,error:失败回掉函数function(msg){....}，beforeSend:请求发送前回调函数`function(msg){….}})
  2. $.get()
    1. 这个函数是基于$.ajax()函数的从这里可以看出$.ajax()函数是属于最底层的函数
    2. 格式：$.get(url,data,callback回掉函数,type服务器返回的数据类型xml，json)
  3. $.post()
    1. 它跟$.get()函数类似也是基于$.ajax()的
  4. $.getJSON(url,data,callback)
    1. 在$.get()方法上进一步封装限制了返回的类型
  5. $.getScipt(url,data,callback)
    1. 在$.get()方法上进一步封装限制了返回的类型
    2. 它的作用是：向服务器发送请求，要求返回的数据为js类型的。这种方法运用用于当页面的js脚本文件很大时，可以通过向服务器发送请求的形式进行加载页面。
  6. $(“div”).load(“url”)
    1. 在$.get()方法上进一步封装了success回掉函数
    2. 这个语句的意思就是向url地址发送请求，将返回的数据显示在div里面

ajax当中json数据格式

  1. 含义

    1. javaScript Object Notationjavascript对象类型格式

  2. 定义格式

    1. var a = {key1:value1,key2:value2}//这里a就是一个json对象

  3. 获取值的方式

    1. 取value1：`a.key1`。取value2:`a.key2`

  4. 在使用json的时候服务器往往会返回这样的数据

    第一种：var v1 =[{name:”tom”,age:18},{name:”zac”,age:20}]

    获取name:”zac”的方式：v1[1].name

    第二种：var v2={name:”zac”,company:{job:engineer}}

    获取”zac”在公司里面的职位： v2.company.job
