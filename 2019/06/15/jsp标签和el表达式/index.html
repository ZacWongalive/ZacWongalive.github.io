jsp标签和el表达式

发表于 2019-06-15 | 分类于 学习-javaweb | | 本文总阅读量 次

jsp标签

  1. 什么是Jsp标签？

    • jsp标签用来替换jsp中的java代码，因为直接在jsp文件当中写java代码不方便Jsp文件维护（比如，将Jsp文件交给美工去修改就很不方便，所以sun公司就制定了jsp标签技术规范.
    • 可以将jsp变迁看作是一个占位符，容器遇到jsp标签之后，会依据标签名找到标签类，然后调用标签类中的方法。
    • 使用Jsp标签，有两个好处：
      • jsp文件很好维护
      • 方便代码的复用（可以将一些常见的代码封装成一个jsp标签）

  2. jstl(javaa standard tag lib java标准标签库)

    1. jstl 是什么？

      • apache开发的一套Jsp标签，后来捐献给了sun,sun将其命名为jstl。

    2. 怎么使用jstl?

      1. 将jstl相关的jar文件拷贝到WEB-INF\lib下。 如果使用javaee5.0就一般不用导包
      2. 使用taglib指令导入相应的标签

    3. jstl当中几个核心的标签

      1. if

        1. 格式<c:if text=””>标签体
          • 在这里if就是一个表亲，需要介绍命名空间和前缀
          • 命名空间：为了区分同名元素而设置的的一个字符串
          • 而如果这个标签在jsp页面中就会有个对应的taglib标签<%@taglib uri=”http://www.tarena.com.cn" prefix=”c”%> uri 就是命名空间 prefix就是前缀(别名)
          • 当text属性值为true时，容器会执行标签体的内容。text属性可以使用el表达式来计算。
          • 这里的话解释以下<c:if>标签的执行流程：当容器看到这个if标签的时候就会根据taglib标签当中指定的前缀和命名规范往jstl.jar中找到相应的标签类。将标签类实例化并执行里面的方法test那块的意思就是如果request当中绑定的user的gender为’f’的话就执行<c:if>当中的内容，显示为男，var和scope那块的话，意思就是将test的属性那块的el表达式的结果作为 request.setatrribute(‘flag’,’el表达式结果true/false’) 绑定起来。所以在判断性别不为’f’的时候，就直接再命名一个标签test的属性值就依据flag来确定
          • scope绑定的范围只能是page,application,session,application

      2. choose

        1. 格式

          • 用来便利集合或者数组。
          • items属性：用来指定要遍历的集合或者数组，可以使用el表达式。
          • var属性：用来指定一个绑定名，绑定范围固定是pageContext.
            • 注：没次从集合或者数组中取一个元素，然后将其绑定导pageContext上，绑定名由var属性来指定。
          • varStatus 用来指定一个绑定名，绑定范围固定是pageContext
            • 绑定值是一个特殊的对象，该对象提供了一些方法，用来获得当前遍历的状态。
            • getIndex():获得当前被遍历的元素的下标（从0开始）。
            • getCount():获得当前是从第几次遍历(从1开始)

        2. forEach标签的示例详细看R中的emplist.jsp和update.jsp文件移除jsp文件当中的java代码

  3. 自定义标签

    1. 如何自定义标签？

      1. 写一个Java类，继承SimpleTagSupport类(简单标签技术类)。

      2. override doTag方法，在该方法里面 编写处理逻辑。

        1. 标签有哪些属性，标签类要有对应的属性
          1. a.要求属性名一直
          2. b.类型要匹配
          3. c.这些属性要有对应的set方法
          4. d.为什么需要set方法呢？ 详细看p25 1:33:18
          5. 这个自定义标签到底是怎么执行的？详细看P25 2:31:29

      3. 描述标签（.tld文件）*就是绑定标签和标签类的对应关系 *

        1. 注意：

          1. tld文件要放在WEB-INF\下。

          2. 的值可以是

            empty:该标签没有标签体
            scriptless：可以有标签体，并且标签体的内容不能出现java代码（<% %>,<%= %>,<%! %>）
            JSP: 可以出现Java代码。但是，只有负载标签技术才支持改制，简单标签技术只支持empty和scriptless.

el表达式

  1. 什么是el表达式？

    • 是一套简单的运算规则，用于给jsp标签的属性赋值，也可以直接输出

  2. el表达式的使用

    1. 访问Bean的属性
      • javabean就是一个java类有属性以及获取该属性的方法
      • 方式一：
        • ${user.name}
          • 如何执行的？ 容器依次会从pageContext,request,session,application中查找(通过调用getAtrribute()方法)绑定名称为”user”的对象，并且调用该对象的”getName()”方法，最后输出getName()方法的返回结果。
          注意：1.el表达式会将获取为Null的值转化成空字符串输出2.如果找不到该对象，则会输出空字符串3.如果要制定查找范围，可以使用pageScope,requestScope,sessionScope,application来指定。
      • 方式二
        • ${user[“name”]} 等价于方式一
        • 注：
          • []里面可以出现绑定名在下图中[str] 中 容器也会依次从pageContext,request,session,application中查找绑定名为str所对应的值
          • []里面可以出现从0开始的下表，用于访问数组中的某个元素
    2. 进行一些简单的运算, 运算的结果可以给jsp的标签的属性赋值，也可以直接输出。
      1. 算数运算
        • + - * / % 注：加号只能求和
      2. 关系运算
        • >,<,<=,>=,==,!=
      3. 逻辑运算
        • && || !
      4. empty运算
        • 用来判断 集合是否为空 或者是否为一个空字符串，如果为空则返回true
        • 格式： ${empty 想要查找的绑定名}
    3. 读取请求参数值(用的不多)
      1. 例如向jsp传入参数 e3.jsp?username=zac
      2. 格式
        1. ${param.username}等价于request.getParameter(“username”)
        2. ${paramValues.username}等价于request.getParameterValues(“username”)
